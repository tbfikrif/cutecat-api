# README

### System dependencies

- Ruby version 3.0.1

- Rails version 6.1.3.2

- PostgreSQL version 12.6

### Configuration

```
bundle install
```

### Database creation

```
rake db:create
```

### Database initialization

```
rake db:migrate
```

### How To Run Development

```
rails server
```

### How To Run Console

```
rails console
```
